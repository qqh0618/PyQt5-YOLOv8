## 二、 项目简介

yolov8作为一个成熟的项目，没有完整的GUI，yolov8与yolov5项目结构有很大不同，因此本项目借鉴优秀的yolov5-pyqt项目，然后进行二次开发，实现yolov8-gui

## 三、注意事项
- 过年期间办公条件有限，仅测试了图片检测及视频检测
- 导入视频或图片后需要点击播放按钮才可以检测（后面会修改）
- 不完善的地方请多多提出意见
